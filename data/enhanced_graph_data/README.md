# Enhanced Graph Data

This directory contains graph data generated from the `enhanced_mock_data` directory. The data is formatted for import into Nebula Graph.

## Data Generation

Generated by: `src/scripts/generate_enhanced_graph_data.py`

Source data: `data/enhanced_mock_data/`

## Data Statistics

- **Person nodes**: 80
- **Company nodes**: 90
- **Contract nodes**: 100
- **Legal event nodes**: 20
- **Transaction nodes**: 60

- **Customer edges**: 73

- **Case-contract edges**: 10

- **Company-transaction edges**: 120

## Import to Nebula Graph

To import this data into Nebula Graph, use:

```bash
# Navigate to project root
cd /path/to/contract-graph

# Import using enhanced_graph_data directory
uv run python src/scripts/nebula_import.py --data-dir enhanced_graph_data
```

## Regenerate Data

To regenerate this data from enhanced_mock_data:

```bash
uv run python src/scripts/generate_enhanced_graph_data.py
```
